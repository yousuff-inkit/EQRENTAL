Application:         Leasing    

 

Process:                Client 

 

Objectives:           This document explains how to create the customer profiles.

 

Prerequisites:       Customer Category , Sales Man

                                                                                                                                                                                               

 

Navigate to:    Operations ->Client Relation -> New Customer

 

Best Practices:

 

Before inducting a new customer into the system, Please confirm that he/she is not an existing customer by  querying on his/her name , mobile number and email address and license number

 

 

Overview

 

Client Profile is create with Minimum information, such his/her  Name , Mobile, Email address and later to update the driver/user of the vehicle leased details such as DOB, License etc. Along with AR Account is created with default service charges of Salik & traffic Fines.

 

 

 

Fields Highlighted is the Mandatory Information to Key in this Tab. In Same New customer there is Tab Named Know Your Customer

 

 

 

Now when you have input all the necessary information the customer profile can be saved in the system by clicking the    Save Button. If you like to discard the entered information click on the  Cancell button . Once the primary information saved , You update the information any time.

Application:         Leasing    

 

Process:                Quotation Followup 

 

Objectives:           Follow-up the quotations sent to customer and Upon acceptance of your proposal to create Lease Application from this Process

 

Prerequisites:       Lease Calculator / Quotation 

                                                                                                                                                                                               

 

Navigate to:    Home->Marketing -> Quotation Follow-up

 

Overview:

 

All Pending quotation will be lined up in this BI report, wherein Sales Agent will chase the customer for his acceptance on your proposal and if any further follow-up is required the sales agent will enter the comments from customer and follow-up on the day customer requested to call or further notification. Below screen is the Menu Map

 

Double Click on the Menu Name  Marketing and the system will list the sub menus, from these Menu Double Click select Lease Quotation Follow

 

 

 

Select the Appropriate Criteria that you wish to see and click on the Button shown in the screen, with given selection in the screen when execute system will list the Quotation pending from customer approval for you In order to create and Lease Application 

 

 

After the Data is Loaded select your quotation for which you would like to create an application by Double clicking on the line selected , In our example one we entered the enquiry->quotation was SAIF with document No.3 it gets pink color . Now click on the Button Application. system takes you automatically to Lease Application Form with all the information Carried forward.

 

 

 

 

 

 

 

 

 

 

 

 

 

A New document is created with all information carried forward from the quotation

1.       Decision to Make the application is for Lease to own or Not  Just by Ticking/ Unticking box Lease to own

2.       The system brings all Lease Value and Lease Per month Value along with 3 Months Advance automated to come in the Advance column , incase if the customer is paying upfront full payment the sales agent is having an option to input the advance ,

3.       No, of instalments

4.       if the customer is willing to have Peace of Mind package CDW you can select the service from the Drop down list in the column of CDW by selecting Gold ,Silver etc.

 

 

Once all the Fields are in place. Click the Save Button to complete the Lease Application Form

Create Lease Application , In put the decision making information for the application and Attach the documents required for approval from the supervisor/manager and finance

 

 

 

 

 

A New document is created with all information carried forward from the quotation

1.       Decision to Make the application is for Lease to own or Not  Just by Tick/ Un-tick box Lease to own

2.       The system brings all Lease Value and Lease Per month Value along with 3 Months Advance automated to come in the Advance column , in case if the customer is paying upfront full payment the sales agent is having an option to change the advance ,

Document attachment.

The Supervisor when he/she logs in the system. All pending for Approval will be listed in the below screen. By clicking on the Icon show below another tab will be opened

 

 

Which has got the details of document pending for approval, it was submitted from which user and date time.
